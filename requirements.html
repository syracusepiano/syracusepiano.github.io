Lesson Requirements:

  • parent or guardian must be at all in-person lessons.
  • method book
    • provided for free digitally
    • client can either print the digital book or use a tablet with notation capability
    • printed book must be double-sided so duets show on facing pages
  • lesson notebook for recording assignments
  • device with videoconferencing software
    • if using Android, install Jitsi Meet app
    • if using iOS, install Jitsi Meet app and please note that the iOS version is usable but buggy
    • if using MacOS, Windows, or Linux, go to meet.jit.si using Chrome or Chromium web browser
    • enter the provided meeting name and password
    • occasionally other students invited to remote lessons to provide performance opportunities
  • metronome
    • must not be installed on the videoconferencing device
    • must be available for all practices and lessons
  • good Internet connection
  • payment
